## What this is

A coding-exam Android app that fetches Taiwan Stock Exchange
(TWSE) open-data and renders a merged stock list. The same screen is
**deliberately implemented three times** — pure XML Views, a Hybrid
(XML shell + Compose rows), and full Jetpack Compose — all driven by **one
shared MVI `StockListViewModel`**.
